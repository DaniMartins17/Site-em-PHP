Bem-vindo

Nossa base de ensino trabalha com diversos cursos, como:

HTML e CSS

O curso é ideal para iniciantes que desejam entender a estrutura e a formatação de páginas web, fornecendo uma base sólida para avançar em desenvolvimento web.

Inteligência Artificial

O curso tem o objetivo de formar profissionais aptos a resolver problemas complexos por meio da Inteligência Artificial.

Google para Educadores

Google para Educadores é uma plataforma que oferece ferramentas e recursos para professores e instituições educacionais.

Python

Python é uma linguagem de programação de alto nível conhecida por sua simplicidade e legibilidade.

Word - Básico

Microsoft Word é um processador de texto amplamente utilizado para criar documentos, cartas, relatórios e outros textos.

Inglês - Avançado

Inglês é a língua franca global, essencial para comunicação em muitas áreas profissionais e acadêmicas.
